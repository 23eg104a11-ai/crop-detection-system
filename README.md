# 🌿 Crop Detection System

## 📌 Description
This project is a simple crop detection system built using Spring Boot and HTML.

## 🛠 Technologies Used
- Java (Spring Boot)
- HTML, JavaScript

## 🚀 Features
- Upload crop image
- Predict crop type
- Displays result

## 📊 Output
Crop Detected: Wheat 🌾

## 🔮 Future Scope
This system can be extended by integrating a Machine Learning model for real-time crop detection.
